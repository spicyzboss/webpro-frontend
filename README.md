# webpro-frontend

Frontend for webpro 2/21 project

## install dependencies

via npm

```bash
npm i
```

via yarn

```bash
yarn
```

## serve with hot reload at localhost:3000

via npm

```bash
npm run dev
```

via yarn

```bash
yarn dev
```
